# 👑 MiniGame - Princess Peach ✨
> Um minigame interativo desenvolvido com tecnologias web, inspirado no universo da Nintendo, focado na personagem Princess Peach. O objetivo é fazer Peach desviar dos obstáculos para alcançar a pontuação máxima.

## 🎮 Demonstração

Clique aqui para jogar

## 🚀 Funcionalidades
Movimentação: Controle a Princess Peach utilizando a tecla "espaço".
Sistema de Pontuação: Acumule pontos ao 
Dificuldade Progressiva: O jogo se torna mais desafiador à medida que você avança.
Responsividade: Jogue no computador ou no celular.

## 🛠️ Tecnologias Utilizadas
As seguintes ferramentas foram usadas na construção do projeto:
- HTML5: Estrutura do jogo e elementos da interface.
- CSS3: Estilização e efeitos visuais.
- JavaScript (ES6+): Lógica do jogo, animações e loop.

## 📦 Como rodar o projeto localmente
Para clonar e executar este aplicativo, você precisará de um navegador moderno e do Git instalado em seu computador.
Bash
## 1. Clone este repositório
$ git clone https://github.com/vivigcsales/MiniGame---Princess-Peach.git

## 2. Acesse a pasta do projeto
$ cd MiniGame---Princess-Peach

## 3. Abra o arquivo index.html no seu navegador
> Dica: Se usar o VS Code, você pode clicar com o botão direito no index.html e selecionar "Open with Live Server"
